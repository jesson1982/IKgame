Events Guide

Stay updated with all Infinity Kingdom events. Learn how to maximize rewards from weekly, daily, and special events.

Weekly Events

Featured

Kingdom Championship

The premier weekly event where kingdoms compete for exclusive rewards and prestigious titles.

  • Schedule: Every weekend (Saturday-Sunday)
  • Requirements: Top kingdoms by power ranking
  • Rewards: 5000 Gems, Legendary Scrolls, Exclusive Title
  • Strategy: Coordinate kingdom-wide effort, focus on objectives

Growth Competition

Compete individually and as an alliance by completing growth tasks throughout the week.

  • Schedule: Monday through Sunday
  • Tasks: Building upgrades, research, troop training
  • Rewards: Growth Packs, Speed-ups, Resources
  • Tip: Save upgrade materials for event week

Daily Events

Dragon Hunt

Team up daily to defeat powerful dragons and earn rare materials for equipment crafting.

  • Reset Time: Daily at server reset
  • Energy Cost: Use all dragon energy during event
  • Rewards: Dragon Essence, Equipment Boxes, Hero XP
  • Strategy: Join alliance rallies for legendary dragons

Daily Quests

Complete daily tasks to earn activity points and unlock milestone reward chests.

  • Reset: 00:00 UTC daily
  • Tasks: Gather resources, train troops, help alliance
  • Rewards: Gold, Resource Packs, Summon Scrolls
  • Goal: Reach maximum activity points every day

Gathering Event

Special event with increased gathering speed and bonus resource tiles appearing on the map.

  • Schedule: Rotating schedule, check in-game
  • Bonus: 2x Gathering Speed
  • Strategy: Prepare armies and heroes beforehand
  • Rewards: Massive resource income potential

Special Events

Limited Time

Seasonal Festivals

Limited-time seasonal events with unique themes, special tasks, and exclusive rewards.

  • Examples: Spring Festival, Halloween, Winter Celebration
  • Duration: Usually 7-14 days
  • Rewards: Exclusive skins, heroes, decorations
  • Strategy: Complete all event tasks for maximum rewards

Summoning Events

Special summoning events with increased rates for epic and legendary heroes.

  • Frequency: Monthly or bi-monthly
  • Bonus: Higher drop rates, guaranteed pulls
  • Strategy: Save scrolls for these events!
  • Reward: Best opportunity to get rare heroes

💡 Golden Rule

NEVER use hero scrolls during non-event periods! Always save for summoning events with bonus rates and guarantees.

Event Maximization Tips

  • Preparation: Stockpile resources and items before events start
  • Calendar Tracking: Keep track of event schedules to plan ahead
  • Priority Tasks: Focus on high-reward tasks first
  • Alliance Coordination: Work with alliance for group event goals
  • Resource Management: Don't waste resources between events
  • Event Synergy: Some events overlap - maximize both simultaneously
